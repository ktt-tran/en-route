En Route — Simplified Product Roadmap
Phase 1 — Navigation MVP (Foundation)

Goal: Build a working trip app that can create and display routes.

Features

✅ Open app
✅ Show current location
✅ Search destinations
✅ Calculate route
✅ Display route on map
✅ Start trip
✅ Save recent trips locally

Tech Stack
React Native + Expo + TypeScript
MapLibre + OpenStreetMap
FastAPI
Valhalla
SQLite (local)
Docker

Result

A user can go:

Current Location
        ↓
Search Destination
        ↓
Get Route
        ↓
View Trip
        ↓
Navigate
