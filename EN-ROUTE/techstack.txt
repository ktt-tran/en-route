Phase 1 (MVP)

This is all you need to build and ship the app.

Layer	Technology	Purpose
Mobile	            React Native + Expo + TypeScript	    Cross-platform iOS/Android app
Navigation	        React Navigation	                    Screen navigation (likely just one main screen)
State Management	Zustand	                                Store current trip, location, route state
Server State	    TanStack Query	                        Cache API responses from your backend
Map Rendering	    MapLibre GL	                            Display the map
Map Data	        OpenStreetMap	                        Free map data
Location	        Expo Location	                        GPS and user location
Search / Geocoding	Nominatim (or a hosted geocoder later)	Convert addresses to coordinates
Backend	            FastAPI (Python)	                    Business logic and APIs
Routing Engine	    Valhalla	                            Calculate routes
Database (Device)	SQLite	                                Recent trips, saved places, settings
Database (Server)	None initially	                        Not needed for the MVP
AI	                None	                                Add later
Authentication	    None	                                No accounts initially
Hosting	            Docker + a cloud VM	                    Host FastAPI and Valhalla
